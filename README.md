# Eddie San Juan — Portfolio

Personal portfolio site for Eddie San Juan, CEO of E.F. San Juan Inc.

Built with Next.js 16, Tailwind CSS 4, Framer Motion, and TypeScript.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Deployed on Railway via GitHub integration. Pushes to `main` trigger automatic redeployment.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS 4
- **Animation**: Framer Motion
- **Language**: TypeScript
- **Hosting**: Railway
